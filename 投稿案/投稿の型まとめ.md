# 投稿の型まとめ

## 基本テンプレ：共感×言語化（5ステップ）

最も汎用性が高いテンプレ。読者の悩みを「命名」して刺す。

```
① 悩みの描写　　→「〇〇してるけど、〇〇してる…」
② 世間の声 vs 現実 →「みんなが〇〇というから頑張ってるけど…」
③ 問いかけ　　　→「その状態、一回でも感じたことありませんか？」
④ 言語化・命名　→『 〇〇型〇〇 』
⑤ 意外性＋現実　→「え、〇〇なのに〇〇なの？！」「でも事実なんです」
```

### 命名の型
`『 〇〇型・〇〇ゼロ発信者 』`

| テーマ | 命名例 |
|---|---|
| ギバーなのにノウハウ集め | 『ギバー型ノウハウコレクター』 |
| リプしてるのに認知ゼロ | 『頑張り型・認知ゼロ発信者』 |
| 学んでるのに動けない | 『完璧主義型・インプット依存者』 |
| DMしてるのに認知ゼロ | 『行動型・認知ゼロ営業マン』 |
| 交流してるのに売れない | 『仲良し型・信頼ゼロ発信者』 |
| いいね返ししてる | 『勤勤型・信頼貯金ゼロ発信者』 |
| 共感リプしてる | 『共感型・価値提供ゼロ発信者』 |
| おはよう投稿してる | 『継続型・信頼貯金ゼロ発信者』 |
| フォロワー増えたのに売れない | 『人気者型・信頼残高ゼロ発信者』 |

## 投稿の型一覧

### 1. 失敗告白型
**用途：** 共感を集めてフォローさせる
**構造：** 失敗の描写 → 気づき → 再起宣言

```
〇〇だった。（数字で見せる）

当時のスペック👇
❌ 〇〇ゼロ
❌ 〇〇ゼロ

→ やめなかった理由
→ 気づき
→ このアカウントでやること🔥
```

---

### 2. 他者の言葉引用型
**用途：** 信頼・実績の証明。反応が取れやすい
**構造：** 引用 → 何をしたか（シンプルに） → 学び

```
〇〇さんに言われた一言。
「〇〇」

別に何もしてない。ただ〇〇しただけ。
・やったこと①
・やったこと②

〇〇で積み重ねた信頼は、思った以上に強かった🔥
```

> **新規垢注意：** 実績がない段階では使いにくい。信頼が溜まってから使う。

---

### 3. 気づき・学びシェア型
**用途：** フォロー促進・保存されやすい
**構造：** 問題提起 → ビフォー → 気づき → 解決策

```
「〇〇」と「〇〇」は全然違う。

✍️ 〇〇→増えた
✍️ 〇〇→増えた
✍️ 〇〇→ゼロのまま

なぜか？→ 原因の一言説明

「〇〇」がゴールじゃない。
「〇〇」がゴールだった💡
```

---

### 4. 行動ルール公開型
**用途：** 保存・シェアされやすい実用投稿
**構造：** ルール提示 → 理由 → 呼びかけ

```
今、自分に課しているルール。

📌 〇〇（禁止事項）
📌 〇〇（やること）
📌 〇〇（マインド）

なぜ〇〇にしたか？→ 理由を一言で

お金じゃなくて、行動で〇〇を作る。
それが今の自分のルール🔥
```

---

### 5. Before/After対比型
**用途：** 変化・成長を視覚的に伝える
**構造：** Before → After → 何を変えたか

```
〇〇前と今、何が変わったか。

【〇〇前】
😞 〇〇
😞 〇〇

【今】
🔥 〇〇
🔥 〇〇

変えたのは「〇〇」だけ。
才能でも、センスでも、お金でもなかった🌱
```

---

### 6. 問いかけ型
**用途：** リプを誘い、交流を作る
**構造：** 共感できる問い → 答え → 学び

```
〇〇してるのに結果が出ない。
そんな経験、ある？

原因は〇〇だった。

❌ 〇〇
⭕️ 〇〇

定義を変えた瞬間、やることが全部クリアになった💡
```

---

### 7. 行動宣言×誘い型
**用途：** フォロー獲得・アカウントのコンセプト提示
**構造：** 宣言 → やること → 誘い

```
今月、〇〇だけを考える。

やること👇
🎁 〇〇
🎁 〇〇
🎁 〇〇

「〇〇」を自分で証明する。

同じ考え方で動きたい人、
フォローしてついてきてほしい🌱
```

---

### 8. 謎かけ型
**用途：** 続きを読ませる・フォロー促進
**構造：** 逆説の問い → 解説 → 真実

```
〇〇で一番やってはいけないこと。

「〇〇すること」を目的にすること。

え、〇〇していいのに〇〇しちゃダメなの？🤔

違う。〇〇自体は悪くない。
問題は、それを"ゴール"にしてしまうこと。

ゴールは「〇〇」。
〇〇は、その通過点にすぎない💡
```

---

### 9. ノウハウ解説型
**用途：** 保存・シェアされやすい。専門性を示す
**構造：** NG例 → OK例 → 違いの一言まとめ

```
〇〇の具体的なやり方。

❌ やってはいけない〇〇👇
・〇〇
・〇〇

⭕️ 〇〇が積まれる〇〇👇
・〇〇
・〇〇

違いは「〇〇かどうか」。

〇〇しようとする〇〇は、もらうだけ。
〇〇を積む〇〇は、与え続ける🔥
```

---

### 10. 新規垢・初投稿型
**用途：** フォロワーゼロからの自己開示
**構造：** 自己開示 → 失敗 → 気づき → このアカウントの約束

```
はじめまして。（または省略）

正直に話す。（または「〇〇だった。」から入る）

😞 〇〇（失敗①）
😞 〇〇（失敗②）
😞 〇〇（失敗③）

→ 気づき（「やり方が間違ってただけ」など）

このアカウントでは〇〇を全部さらけ出す。
きれいごとは言わない。

同じ場所でつまずいてる人に届いてほしい🌱
フォローしてくれると嬉しいです。
```

---

## 使い分けの目安

| 状況 | 使うべき型 |
|---|---|
| 新規垢・初投稿 | 失敗告白型 / 新規垢初投稿型 |
| フォロワー少ない段階 | 問いかけ型 / 行動ルール公開型 |
| 信頼が溜まってきた | 他者の言葉引用型 / 断言型 |
| バズらせたい | 共感×言語化（命名）型 |
| 保存・シェアされたい | ノウハウ解説型 / Before/After型 |
| フォロー獲得したい | 行動宣言×誘い型 / 謎かけ型 |
