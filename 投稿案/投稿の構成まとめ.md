# 投稿の構成の型

---

## 構成の3要素

すべての投稿はこの3つで成り立つ。

```
① フック　　→ 最初の1〜2行。読むか捨てるかを決める
② ボディ　　→ 主張・根拠・ストーリー
③ CTA　　　→ 次の行動（フォロー・リプ・保存）を促す
```

---

## 構成の型 一覧

---

### 構成A｜問題提起→共感→解決

読者の悩みから入り、答えを渡す。

```
【フック】
〇〇してるのに、結果が出ない。

【ボディ】
原因はこれだった👇

❌ 〇〇（間違い）
⭕️ 〇〇（正解）

違いは「〇〇かどうか」だけ。

【CTA】
定義を変えた瞬間、全部クリアになった💡
```

**向いているテーマ：** 集客・発信・信頼構築の「気づき」

---

### 構成B｜結論→理由→具体例

最初に答えを言う。読み飛ばされにくい。

```
【フック＝結論】
〇〇は、〇〇より〇〇だ。

【ボディ＝理由】
なぜか？

理由は〇〇。

【具体例】
実際にやったこと👇
・〇〇
・〇〇

【CTA】
これを知ってから、発信が変わった🔥
```

**向いているテーマ：** ノウハウ・持論・主張系

---

### 構成C｜共感→命名→現実

読者の状態を言語化して「名付ける」。刺さりやすい。

```
【フック＝共感】
〇〇してるけど、〇〇してる…
みんなが〇〇というから動いてるけど…

【ボディ＝問いかけ】
その状態、一回でも感じたことありませんか？

その状態を言語化すると…

【命名】
『 〇〇型〇〇 』

【CTA＝現実】
え、〇〇なのに〇〇なの？！
でも、事実なんです😢
```

**向いているテーマ：** 悩みの言語化・ターゲットへの刺さり

---

### 構成D｜ストーリー（起承転結）

読み物感が強く、最後まで読まれやすい。

```
【起＝フック】
〇〇だった。（過去の状態を一言で）

【承＝展開】
〇〇をした。でも〇〇だった。

【転＝気づき】
気づいたのは、〇〇だということ。

【結＝CTA】
今は〇〇になった。
同じ状況の人に届いてほしい🌱
```

**向いているテーマ：** 失敗談・成長ストーリー・自己開示

---

### 構成E｜Before/After

変化を視覚的に見せる。スキャンされやすい。

```
【フック】
〇〇前と今、何が変わったか。

【Before】
😞 〇〇
😞 〇〇
😞 〇〇

【After】
🔥 〇〇
🔥 〇〇
🔥 〇〇

【転＝何を変えたか】
変えたのは「〇〇」だけ。

【CTA】
才能でも、お金でもなかった🌱
```

**向いているテーマ：** 成長・変化・行動の結果

---

### 構成F｜リスト型

保存・シェアされやすい。実用性が高い。

```
【フック】
〇〇のためにやること、全部まとめた。

【リスト】
📌 〇〇
📌 〇〇
📌 〇〇
📌 〇〇
📌 〇〇

【まとめ一言】
全部やって気づいたのは、〇〇だということ。

【CTA】
保存して使ってほしい🔥
```

**向いているテーマ：** ノウハウ・チェックリスト・まとめ

---

### 構成G｜対比型

2つを並べて「選ばせる」。読者に当事者意識を持たせる。

```
【フック】
同じ〇〇でも、定義が違うと結果が変わる。

【対比A】
【〇〇な〇〇】
・目的：〇〇
・手段：〇〇
・結果：〇〇

【対比B】
【〇〇な〇〇】
・目的：〇〇
・手段：〇〇
・結果：〇〇

【CTA＝問いかけ】
あなたはどっちで動いてる？🔥
```

**向いているテーマ：** 考え方の違い・正解 vs 不正解

---

### 構成H｜引用→行動→学び

他者の言葉を借りて信頼を作る。

```
【フック＝引用】
〇〇さんに言われた一言。
「〇〇」

【行動】
やったこと👇
✅ 〇〇
✅ 〇〇

【学び】
〇〇で積み重ねた〇〇は、
思った以上に強かった🔥
```

**向いているテーマ：** 信頼・行動・実績の証明
> 新規垢では使いにくい。信頼が溜まってから使う。

---

### 構成I｜謎かけ→解説→真実

「え？」で読ませる。最後まで離脱させない。

```
【フック＝謎かけ】
〇〇で一番やってはいけないこと。
「〇〇すること」を目的にすること。

【疑問】
え、〇〇なのに〇〇しちゃダメなの？🤔

【解説】
違う。〇〇自体は悪くない。
問題は、それを"ゴール"にしてしまうこと。

【真実＝CTA】
本当のゴールは「〇〇」。
〇〇は、その通過点にすぎない💡
```

**向いているテーマ：** 逆説・常識を覆す主張

---

### 構成J｜宣言×誘い

アカウントのコンセプトを示してフォローさせる。

```
【フック＝宣言】
今月、〇〇だけを考える。

【やること】
🎁 〇〇
🎁 〇〇
🎁 〇〇

【理由・信念】
「〇〇が最強だ」と証明する。

【CTA＝誘い】
同じ考えで動きたい人、
フォローしてついてきてほしい🌱
```

**向いているテーマ：** 初投稿・方針転換・月初め宣言

---

## フックの型まとめ

投稿の命運は最初の1〜2行で決まる。

| フックの型 | 例文 |
|---|---|
| 数字で見せる | 「noteの売上、400円だった。」 |
| 断言する | 「Threadsは仲良しごっこの場じゃない。」 |
| 問いかける | 「頑張ってるのに、結果が出ない。そんな経験、ある？」 |
| 逆説で驚かす | 「交流してるのに、信頼ゼロ。」 |
| 引用で始める | 「先生に言われた一言。『〇〇』」 |
| 悩みを代弁する | 「毎日投稿してるけど、全然売れない…」 |
| Before提示 | 「半年前と今、何が変わったか。」 |

---

## CTAの型まとめ

| CTAの型 | 例文 |
|---|---|
| フォロー誘導 | 「フォローしてついてきてほしい🌱」 |
| 共感問いかけ | 「同じ経験ある人、リプで教えてほしい」 |
| 保存誘導 | 「保存して使ってほしい🔥」 |
| 宣言で締める | 「やめない。やりきる🔥」 |
| 読者への問い | 「あなたはどっちで動いてる？」 |
| 届けたい人を指定 | 「同じ場所でつまずいてる人に届いてほしい🌱」 |

---

## 構成の使い分け早見表

| 目的 | 使うべき構成 |
|---|---|
| 初投稿・自己開示 | D（ストーリー）/ J（宣言×誘い） |
| フォロワー増やす | C（共感→命名）/ I（謎かけ） |
| 信頼を積む | A（問題提起→解決）/ H（引用） |
| 保存・シェアされる | F（リスト）/ E（Before/After） |
| 主張を通す | B（結論→理由）/ G（対比） |
| リプをもらう | A（問題提起）/ G（対比・問いかけ締め） |
